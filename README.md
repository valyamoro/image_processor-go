Демонстрация высокопроизводительной системы обработки изображений на Go с использованием пула воркеров (Worker Pool) и приоритетной очереди задач.

Worker Pool Pattern — фиксированное количество горутин для контроля параллелизма
Fan-out / Fan-in — распределение задач и сбор результатов
Priority Queue — обработка старых файлов в первую очередь
Real-time Monitoring — отслеживание состояния системы
Graceful Shutdown — корректное завершение работы
